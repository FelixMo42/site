ABOUT

Super Link is a website created by Felix Moses, Milo Moses, Addison Sutton, Parker Sutton for HS Hacks IV. Its a web application that lets you link variable together in a web and it will automatically solve for all missing variable it can. The backend was build using Flask (A python library for web site back end development), letting it handle arbitrarily large numbers.
